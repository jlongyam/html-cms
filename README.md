# [html-cms](https://github.com/jlongyam/html-cms)

Tiny CMS in single file plain html

## Goal

- Code playground html, js, css
- Code preview
- File snippet
- Keep `<mark>`
- Small size, `8 KB`
- Configurable
- Mobile friendly
- Data stored in EOF
- [Demo](https://jlongyam.github.io/html-cms/display.html)
